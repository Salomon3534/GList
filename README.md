GList: Tu Diario de Entrenamiento Inteligente

GList es una aplicación de gestión de entrenamiento enfocada en la simplicidad y el análisis de datos para usuarios que buscan un control riguroso de su progreso en el gimnasio. Su interfaz permite registrar cada sesión mediante un sistema de cadenas de datos estructuradas, facilitando la visualización de la evolución a largo plazo.
Características Principales

    Registro Estructurado: Utiliza un formato compacto (Peso-Repeticiones-Fecha) que permite añadir múltiples series de forma rápida, ideal para no perder el ritmo durante el entrenamiento.

    Visualización de Progreso: La aplicación transforma tus registros en gráficos e indicadores de rendimiento, permitiéndote identificar fases de progresión lineal, estancamiento o picos de fuerza.

    Gestión de Historial: Permite almacenar y consultar meses de entrenamiento para ajustar las cargas de trabajo de manera inteligente (sobrecarga progresiva).

    Interfaz Minimalista: Diseñada para evitar distracciones, priorizando la entrada de datos eficiente y la lectura clara de los resultados.
